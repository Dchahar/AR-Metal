# AR-Metal

A custom rendering augmented reality using Metal.

## Features

- Plane detection
- Metal vertex and fragment shaders

## Development

**Technologies**

- [Swift](https://swift.org/)
- [Metal](https://developer.apple.com/metal/)
- [ARKit](https://developer.apple.com/augmented-reality/)

**Requirements**

- iOS device
- Xcode 12 or higher
- iOS 13 or higher
